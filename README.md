# sound-truncate
## Truncate sound files using ffmpeg
I wrote this to truncate some ©-free [sound effect files from pixabay](https://pixabay.com/sound-effects/).

This implementation uses NodeJs and ffmpeg.  And I run this on Windows 11, wsl. Some notes on how to set things up (for my own memory):
* wsl --version
* wsl --update
* wsl -u root
* sudo apt update
* sudo apt install ffmpeg
* node --version
